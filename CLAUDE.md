# Claude

Baseline configuration for Claude Code operating in this workspace.

## Workspace Overview

This repository is a personal life management system. It tracks tasks, stores knowledge, drafts content, and automates daily routines through AI-assisted workflows.

## Key Files

- `SOUL.md` — AI personality and behavioral constraints
- `AGENTS.md` — Operational rules and protocols
- `USER.md` — Owner preferences and context
- `BOOTSTRAP.md` — Initial setup and onboarding guide

## Directory Structure

```
me/

├── briefings/             # Daily information summaries
├── notes/                 # Writing drafts and knowledge base
│   ├── articles/          # Long-form article drafts
│   └── journal/           # Personal journal entries
├── tasks/                 # Task tracking supplements
└── resources/             # Reference materials and templates
```
