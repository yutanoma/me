# Skill: Weekly Review

A Claude Code skill for conducting a structured weekly reflection and planning session.

## Trigger

`/weekly-review`

## Description

Reviews the past week's completed tasks, briefings, and journal entries, then helps plan the coming week.

## Steps

1. Summarize closed GitHub Issues from the past 7 days
2. Review briefings from the past week
3. Identify patterns and wins
4. Draft next week's priorities as new Issues
5. Write a weekly review note to `notes/journal/`

## Output Format

```markdown
# Weekly Review: Week of YYYY-MM-DD

## What I accomplished
- ...

## What I learned
- ...

## Next week's priorities
- [ ] ...
```
