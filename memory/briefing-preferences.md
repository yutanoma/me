# Briefing Preferences

- When running `/morning-briefing`, always:
  - Deduplicate against recent `briefings/*.md` before finalizing.
  - Save the output markdown to `briefings/YYYY-MM-DD.md`.

- Treat `briefings/` as the canonical archive of prior briefings.
- Do not claim briefing history is "not stored" if entries exist in `briefings/`.
